- [x] 1.1 移除現有的 Tabs 切換結構，改為緊湊的單頁網格佈局（左側掃描與設備清單，右側連線控制與測試結果）。

- [x] 1.5 將原本大區塊的互動日誌縮減為畫面底部的「單行狀態跑馬燈」。

- [x] 2.1 修改 `app.js`，實作「⚙️ 進階設定」Modal 的開啟、關閉與外部點擊關閉邏輯。
